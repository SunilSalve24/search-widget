# search-widget

We have used :- 
  - Angular v1.6.4
  - Bootstrap
  - HTML5
  - Scss
  - Gulp
  - Bower
  - Node js (Express Server)
  
Run Application :- 

Run following command in project root folder 
  - Run npm install 
  - bower install
  - gulp sass
  - node app.js

Please load http://localhost:8282 URL
